Quick install

Clone, install dependencies, and scan any public or private repo in under a minute.

$ git clone https://github.com/OmarRao/secure-scope.git $ cd secure-scope $ pip install -r requirements.txt $ python securescope.py --repo https://github.com/yourorg/yourrepo

See the User Guide for Docker, CI/CD, Kubernetes, and all CLI flags.

What's new in v10.0.0

The biggest release yet — eight major capabilities added.

🌐

Polyglot scanning

npm audit, cargo audit, govulncheck, and bundler-audit run automatically across every ecosystem found in the repo.

🗄️

SQLite persistence & trend tracking

The --use-db flag persists all scans and findings for historical trend analysis across runs.

🚨

SLA breach alerting via Slack

Configure SLA windows per severity; SecureScope fires Slack alerts automatically when deadlines are breached.

🎫

Jira Cloud auto-ticket creation

HIGH and CRITICAL findings automatically generate Jira issues with full context via the --jira-* flags.

📄

PDF report export

The --pdf flag renders a polished PDF report via weasyprint — ready for stakeholders and auditors.

🔑

GitHub App authentication

Use --github-app-id and --github-app-key for installation-level auth on private repositories at scale.

📐

Custom Semgrep rules

Ship your own rules in rules/ — Flask security, secrets, and API security bundles included out of the box.

☸️

Kubernetes / Helm deployment

Production-ready k8s/ manifests and a helm/ chart for parameterised cluster deployments.
